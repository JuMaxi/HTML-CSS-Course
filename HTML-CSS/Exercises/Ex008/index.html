Principais formatacoes

Negrito ou Destaque

Nesta frase temos o uso de um termo em Negrito usando a tag B(nao semantica)

Nesta frase tenho um termo em Destaque usando a tag STRONG(Semantica)

Italico ou Enfase

Nesta frase temos um termo em Italico, usando a tag I (nao Semantica)

Nesta frase temos um termo em Enfase, usando a tag EM (Semantica)

Texto Marcado

Podemos tambem criar um texto marcado usando a tag MARK

E no outro paragrafo temos temos outro texto marcado no final, ou seja, nao colocamos o style do paragrafo anterior, por isso permanece a cor padrao amarelo, mas usando uma tag Style em Head, e possivel alterar as cores de todas as tag Mark que estao na cor padrao amarelo, como nesse exemplo que alterei para um azul(aqua).

Texto Grande e Pequeno

Estamos criando um texto grande e um texto pequeno neste paragrafo.

Texto Deletado

Podemos marcar um texto como excluido para indicar que ele deve ser lido, mas nao considerado.

Texto Inserido

Podemos marcar um texto como inserido para dar uma enfase e indicar que ele foi adicionado depois.

Existe tambem o sublinhado(inserido) com a Tag U (nao Semantica)

Texto Sobrescrito

Para inserir coisas do tipo x20+3

Texto Subscrito

Para inserir coisas do tipo H2O
